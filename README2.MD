## Workflow Steps

### Initial Setup
```bash
git clone https://github.com/your-username/ai-startup-website.git
cd ai-startup-website
```
### Tom's Navigation Update
   1. Create branch

      ```bash
      git checkout -b update-navigation
      ```
   2. Edit index.html
      <nav>Navigation Bar Content</nav>
   3. Push changes

      ```bash
      git add index.html
      git commit -m "Add navigation bar"
      git push origin update-navigation
      ```
### Jerry's Contact Info
   1. Sync with main:

      ```bash
      git checkout main
      git pull origin main
      ```
   2. Create branch:

      ```bash
      git checkout -b add-contact-info
      ```
   3. Edit index.html:

      <footer>Contact: email@example.com</footer>

## Commands Cheatsheet
   ```bash
   # Compare branches
   git diff main..update-navigation
   
   # Check merge conflicts
   git status
   
   # Update local repo
   git fetch --all
   git pull origin main
   ```

## Contribution Guide
   1. Fork the repository
   2. Create your feature branch (git checkout -b feature/AmazingFeature)
   3. Commit your changes (git commit -m 'Add some feature')
   4. Push to the branch (git push origin feature/AmazingFeature)
   5. Open a Pull Request
